Códigos de acceso

Genera un código único por cada persona de confianza y entrégaselo tú mismo (en persona o por WhatsApp). Sin un código válido, nadie puede registrar una apuesta — esto evita que alguien se registre haciéndose pasar por otra persona. Si alguien ya te dio la mitad del pago, puedes "apartarle" un código con su nombre — el sistema exigirá que la persona escriba ese mismo nombre en el formulario para poder usarlo, así nadie más puede quitárselo.

Apuestas pendientes de verificar

Verifícalas solo después de confirmar que la persona ya te pagó. Mientras estén pendientes, no cuentan para el bote ni para el reparto.

Registrar resultado final

Una vez que termine el partido, marca quién ganó. Esto calculará el reparto automáticamente. Ciérralo antes para evitar apuestas de último minuto.

Iniciar partido nuevo

Usa esto cuando quieras armar la quiniela para el SIGUIENTE partido. Limpia el marcador (bote, apuestas activas, resultado) para empezar de cero, pero conserva para siempre el historial de apuestas y las estadísticas de cada usuario registrado. Después de darle clic, ve a "Configurar el partido" arriba para elegir los nuevos equipos.

Zona de reinicio

Borra TODAS las apuestas (incluyendo el historial de todos los usuarios), todos los códigos de acceso y el resultado registrado. Esta acción NO se puede deshacer. Úsalo solo para pruebas antes de que arranque la quiniela real — no lo uses una vez que la gente ya tenga historial que quieras conservar.
